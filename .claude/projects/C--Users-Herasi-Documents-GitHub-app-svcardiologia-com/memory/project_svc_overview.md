Building a mobile-first PWA for Sociedad Venezolana de Cardiologia (SVC). Dark luxury aesthetic inspired by high-end fintech apps. 7 modules built sequentially: Base, Auth, Members, Payments, Events, Tickets+QR, QR Scanner, Admin Panel.

**Why:** SVC needs a modern digital presence for managing memberships, events, and payments. Must run on cPanel shared hosting (no Node.js).

**How to apply:** All design decisions should favor the dark luxury aesthetic with red (#D11039) as brand color. Stack is HTML+CSS+JS+GSAP for frontend, PHP 8+MySQL for backend. No frameworks — vanilla everything.
